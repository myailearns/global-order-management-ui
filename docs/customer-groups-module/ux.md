CUSTOMER GROUPS MODULE UX

NAVIGATION
Main menu item: Customer Groups

PAGE 1: GROUP LIST
Sections:
- Header with Create Group button.
- Filter bar: search, status.
- Table:
  - Group Name
  - Description
  - Members
  - Status
  - Updated At
  - Actions (View, Edit, Deactivate)

PAGE 2: GROUP DETAIL
Sections:
- Group summary card.
- Members list with search.
- Add members panel:
  - Add by selecting customers
  - Bulk add via multi-select
- Remove member actions.

CROSS-MODULE ENTRY POINTS
- From Customers list: Assign to Group quick action.
- From Customer detail: Add/Remove group membership.

EMPTY STATES
- No groups created yet.
- Group has no members.
- No customer match for search.

VALIDATIONS
- Name required and unique.
- Prevent duplicate assignment feedback.
- Confirm before deactivation/delete.

FUTURE CAMPAIGN READINESS (PLACEHOLDERS)
- Audience estimate widget.
- "Use this group in campaign" CTA (disabled until campaign module).

MOBILE UX
- Card-based group list.
- Sticky bottom action for Add Members.
