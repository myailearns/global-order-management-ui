COURIER MODULE UX

UX PRINCIPLE
Keep courier dispatch operations simple and auditable.

NAVIGATION (SAME APP)
- Courier Partners
- Courier Analytics

--------------------------------------------------
1. COURIER PARTNERS SCREEN
Columns:
- Name
- Contact Person
- Contact Phone
- Service Areas
- Status
- Dispatched (range)
- Delivered (range)
- Failed or Returned (range)

Actions:
- Add Partner
- Edit Partner
- Activate or Deactivate
- View Performance

--------------------------------------------------
2. ORDER DISPATCH UX
- Select courier partner from dropdown.
- Capture tracking number, AWB number, consignment note, ETA.
- Show courier chip in order list and detail views.

--------------------------------------------------
3. COURIER ANALYTICS UX
Cards:
- Total dispatched
- In transit
- Delivered
- Failed or Returned

Filters:
- Date range
- Courier partner
- Status

--------------------------------------------------
4. EMPTY STATE
- No courier partners configured.
- CTA: Add first courier partner.
