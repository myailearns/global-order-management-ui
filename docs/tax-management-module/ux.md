Tax management UX

ADMIN MASTER SETUP FLOW
1. Open Tax Profiles.
2. Create tax profile (name, mode, rate, inclusive, HSN code).
3. Activate profile.
4. Map profile to product group.
5. Validate tax preview on sample amount.

ORDER MODULE BEHAVIOR
Order user should not type tax manually in normal flow.

Tax is shown as derived value from mapped Tax Profile.

Manual override (if enabled) must ask reason and capture audit log.

VALIDATION MESSAGES
Group tax profile missing.

No active tax profile found for mapping.

Tax profile inactive for current effective date.

India-first message:
Current rollout supports GST-only tax mode.
